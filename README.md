International Debt Statistics Data

Description:
This repository includes a detailed collection of global external debt statistics sourced directly from the World Bank’s International Debt Statistics (IDS) database. The dataset provides a wide range of information that reflects how countries around the world borrow and manage debt owed to foreign lenders. It covers long-term external debt figures for low- and middle-income nations, including annual data on debt stocks and flows, debt composition, and related financial indicators. The data includes public and publicly guaranteed debt, as well as private non-guaranteed debt, and highlights key metrics such as debt ratios, external debt values, and creditor breakdowns that are essential for analyzing borrowing patterns across countries and regions. This information is valuable for researchers, policy makers, economic analysts, and anyone interested in understanding global debt dynamics and trends.

Data Source:
All the data included in this repository has been collected from the World Bank’s International Debt Statistics database, which is one of the most trusted and comprehensive sources of external debt information worldwide. The IDS Database provides yearly statistics on external debt for reporting countries based on standardized reporting to the World Bank’s Debtor Reporting System (DRS). The World Bank regularly updates and maintains this database, ensuring both the reliability and relevance of the information. The dataset covers multiple decades of data and is updated annually, making it particularly useful for trend analysis and long-term research on global debt patterns
